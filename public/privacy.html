Privacy Policy

Last updated: 6 June 2026

Traent Hub Visual Tool is used to create and edit visual slide configurations.

Data processed

When a user creates a visual session, the tool may receive and store:

  • slide titles;
  • slide body copy;
  • visual configuration settings;
  • layout choices;
  • brand settings;
  • generated visual config JSON.

The tool does not require users to submit personal data. Users should avoid entering sensitive personal information into slide content.

How data is used

Data is used only to:

  • create editable visual sessions;
  • load saved visual configurations;
  • allow users to edit and export visual outputs.

We do not sell user data.

Data sharing

Data is not shared with third parties except for infrastructure providers used to host and operate the service.

Storage

Visual configuration sessions may be stored to allow users to reopen and edit them.

Sessions are identified by unguessable session URLs. Anyone with a session URL may be able to access that session.

Security

Session creation may require an API key. Session URLs should be treated as private links.

Data deletion

contact@traenthub.com
